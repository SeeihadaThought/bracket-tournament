# Bracket

이미지로 만드는 토너먼트 도구입니다. 좋아하는 캐릭터, 음식, 영화 포스터 등 이미지 파일을 업로드하고, 둘 중 하나를 골라가며 최후의 1개를 가립니다.

## 사용법

1. 라운드 크기 선택 (4강 ~ 128강)
2. 이미지 파일이나 폴더 업로드
3. 시작 버튼 클릭
4. 매 매치마다 마음에 드는 쪽을 선택

## 기능

- 한국어 / English / 日本語 지원
- 다크/라이트 테마
- GIF 호버 재생 (한 번에 하나만, 성능 최적화)
- 부전승 / 예선전 자동 처리
- 폴더 구조 유지 업로드
- IndexedDB로 대용량 이미지 저장
- 모바일 반응형
